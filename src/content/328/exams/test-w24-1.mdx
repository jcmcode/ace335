## Problem 1: Topology, Interior, Closure, Hausdorff

<Problem title="Topology, Interior, Closure, Hausdorff">
Let us consider the topological space $(X, \mathscr{T})$, where $X = (-3, 3) = \{x \in \mathbb{R} : -3 < x < 3\}$ and $\mathscr{T} \subset 2^X$ is defined as
$$\mathscr{T} = \{(-t, t) : 0 < t \leq 3\} \cup \{\varnothing\}.$$

(a) Verify that $\mathscr{T}$ is a topology on $X$.

(b) Consider the subsets $A, B \subset X$, where $A = (0, 2)$ and $B = \{1\}$. Find the interior of $A$ and the closure of $B$, according to the topology $\mathscr{T}$.

(c) Is $(X, \mathscr{T})$ a Hausdorff topological space? Justify your answer.
</Problem>

<Solution>
**Part (a):** We must verify the three axioms of a topology.

*Axiom 1:* $\varnothing \in \mathscr{T}$ by definition, and $X = (-3, 3) = (-t, t)$ with $t = 3$, so $X \in \mathscr{T}$.

*Axiom 2 (arbitrary unions):* Let $\{(-t_\alpha, t_\alpha)\}_{\alpha \in A}$ be any family of elements of $\mathscr{T}$ (excluding $\varnothing$, which doesn't affect unions). Let $t^* = \sup_{\alpha} t_\alpha$. If the family is empty, the union is $\varnothing \in \mathscr{T}$. Otherwise, $0 < t^* \leq 3$, and
$$\bigcup_{\alpha} (-t_\alpha, t_\alpha) = (-t^*, t^*) \in \mathscr{T}.$$
(If $t^* \leq 3$ and attained or not, this interval equals $(-t^*, t^*)$ with $0 < t^* \leq 3$.)

*Axiom 3 (finite intersections):* For $(-t_1, t_1), (-t_2, t_2) \in \mathscr{T}$,
$$(-t_1, t_1) \cap (-t_2, t_2) = (-\min(t_1, t_2), \min(t_1, t_2)) \in \mathscr{T}.$$
Intersection with $\varnothing$ gives $\varnothing \in \mathscr{T}$.

Therefore $\mathscr{T}$ is a topology on $X$.

**Part (b):**

*Interior of $A = (0,2)$:* $\text{int}(A)$ is the largest open set contained in $A$. The open sets are $\varnothing$ and $(-t, t)$ for $0 < t \leq 3$. Any $(-t, t)$ contains negative numbers, so is not contained in $(0, 2)$. Hence the only open set $\subseteq A$ is $\varnothing$, so $\text{int}(A) = \varnothing$.

*Closure of $B = \{1\}$:* The closed sets are complements of open sets:
$X \setminus \varnothing = X = (-3, 3)$ and $X \setminus (-t, t) = (-3, -t] \cup [t, 3)$ for $0 < t \leq 3$.

We need the smallest closed set containing $\{1\}$. We need $1 \in X \setminus (-t, t)$, i.e., $1 \geq t$ or $1 \leq -t$. Since $t > 0$, we need $t \leq 1$. The smallest such closed set is obtained by taking $t = 1$:
$$\overline{B} = (-3, -1] \cup [1, 3).$$

**Part (c):** $(X, \mathscr{T})$ is **not Hausdorff**. Take $x = 0$ and $y = 1$. Every nonempty open set has the form $(-t, t)$ with $0 < t \leq 3$, which always contains $0$. So any open neighborhood of $y = 1$ also contains $x = 0$, meaning we cannot separate $0$ and $1$ by disjoint open sets.
</Solution>

## Problem 2: Cluster Points, Neighbourhoods, Limits

<Problem title="Cluster Points, Neighbourhoods, Limits">
In this question, the real line $\mathbb{R}$ is assumed to be equipped with the Euclidean topology. Let $D \subset \mathbb{R}$ be given by $D = \mathbb{Q} \cap (0, +\infty)$.

(a) Find $D'$, i.e. the set of all cluster points of $D$.

(b) Is $D$ a neighbourhood of the point $\tfrac{3}{4}$? Justify your answer.

Now let $f : D \to \mathbb{R}$ be a function and let $c \in D'$. Suppose that $\lim_{x \to c} f(x) = 5$. For each of the following statements, say whether it is true or false.

(c1) If $c \in D$, then $f$ is continuous at $c$ only if $f(c) = 5$.

(c2) $5$ must be a cluster point of $f(D)$.
</Problem>

<Solution>
**Part (a):** $c \in D'$ means every neighbourhood of $c$ contains a point of $D \setminus \{c\}$. Since $\mathbb{Q}$ is dense in $\mathbb{R}$, any open interval around a point $c \geq 0$ contains rationals different from $c$, and for $c > 0$ these rationals can be taken positive (for $c = 0$, any open interval $(-\varepsilon, \varepsilon)$ contains positive rationals). For $c < 0$, take $(c - \varepsilon, c + \varepsilon)$ with $\varepsilon = |c|/2$; this contains no positive numbers, so $c \notin D'$.

Therefore
$$D' = [0, +\infty).$$

**Part (b):** $D$ is a neighbourhood of $\tfrac{3}{4}$ iff there exists an open set $U$ with $\tfrac{3}{4} \in U \subseteq D$. In the Euclidean topology, any open set containing $\tfrac{3}{4}$ contains an interval $(\tfrac{3}{4} - \varepsilon, \tfrac{3}{4} + \varepsilon)$, which contains irrationals. But $D \subseteq \mathbb{Q}$ contains no irrationals. Hence **$D$ is not a neighbourhood of $\tfrac{3}{4}$**.

**Part (c1):** **TRUE.** If $f$ is continuous at $c$ and $c \in D \cap D'$, then $\lim_{x \to c} f(x) = f(c)$. Given the limit equals $5$, we must have $f(c) = 5$.

**Part (c2):** **FALSE.** A counterexample: let $f(x) = 5$ for all $x \in D$. Then $\lim_{x \to c} f(x) = 5$, but $f(D) = \{5\}$, which has no cluster points. So $5$ is not a cluster point of $f(D)$.
</Solution>

## Problem 3: Connectedness in R^2

<Problem title="Connectedness in R^2">
In this question, $\mathbb{R}^2$ is assumed to be equipped with the Euclidean topology.

(a) Let $S = \{(x, y) \in \mathbb{R}^2 : x^2 y = 4\}$. Show that $S$ is disconnected.

For each of the following statements, say whether it is true or false. If true, provide a short proof; if false, provide a counterexample.

(b1) Every open subset of $\mathbb{R}^2$ is connected.

(b2) Every nonempty finite subset of $\mathbb{R}^2$ is disconnected.
</Problem>

<Solution>
**Part (a):** Note $S = \{(x, y) : y = 4/x^2, x \neq 0\}$. Define
$$U = \{(x, y) \in \mathbb{R}^2 : x < 0\}, \quad V = \{(x, y) \in \mathbb{R}^2 : x > 0\}.$$
These are open in $\mathbb{R}^2$ (half-planes). Their intersections with $S$ are:
- $S \cap U$: points with $x < 0$, nonempty (e.g., $(-1, 4) \in S$).
- $S \cap V$: points with $x > 0$, nonempty (e.g., $(1, 4) \in S$).
- $(S \cap U) \cap (S \cap V) = \varnothing$ since $U \cap V = \varnothing$.
- $(S \cap U) \cup (S \cap V) = S$ since $x = 0$ gives $0 \cdot y = 0 \neq 4$, so $x = 0 \notin S$.

Thus $\{S \cap U, S \cap V\}$ is a separation of $S$ in the relative topology, so $S$ is disconnected.

**Part (b1):** **FALSE.** Counterexample: $U = B((-1,0), 1) \cup B((1, 0), 1)$ (union of two disjoint open balls) is open but disconnected.

**Part (b2):** **FALSE.** Counterexample: $\{(0, 0)\}$ is a nonempty finite subset of $\mathbb{R}^2$, and a singleton is always connected (any cover by disjoint open sets must place the single point in exactly one piece, so the other piece in the separation is empty).
</Solution>

## Problem 4: Completeness and Baire Category

<Problem title="Completeness and Baire Category">
(a) Let $(X, d)$ be a metric space. Define what it means for $(X, d)$ to be *complete*.

(b) State the Baire Category theorem.

(c) Let $(X, d)$ be a metric space. Suppose that $A \subset X$ is a nonempty meagre set. Use the Baire Category Theorem to show that $A$ is not comeagre.
</Problem>

<Solution>
**Part (a):** $(X, d)$ is *complete* if every Cauchy sequence in $X$ converges to a point of $X$. That is, for every sequence $(x_n) \subseteq X$ such that for all $\varepsilon > 0$ there exists $N \in \mathbb{N}$ with $d(x_n, x_m) < \varepsilon$ for all $n, m \geq N$, there exists $x \in X$ with $\lim_{n \to \infty} d(x_n, x) = 0$.

**Part (b):** *Baire Category Theorem:* Let $(X, d)$ be a complete metric space. Then the intersection of every countable collection of open dense subsets of $X$ is dense in $X$. Equivalently, $X$ is not a countable union of nowhere dense sets (i.e., $X$ is not meagre in itself).

Another equivalent form (often used in this course): if $(X, d)$ is complete and nonempty, then $X$ is not meagre.

**Part (c):** Recall that $A$ is *meagre* if it is a countable union of nowhere dense sets, and $A$ is *comeagre* if $X \setminus A$ is meagre.

Assume for contradiction that $A$ is both meagre and comeagre. Then $X \setminus A$ is meagre, so both $A$ and $X \setminus A$ are meagre. Hence
$$X = A \cup (X \setminus A)$$
is a union of two meagre sets. The countable union of meagre sets is meagre, so $X$ is meagre.

But by the Baire Category Theorem (statement given), a complete nonempty metric space is not meagre in itself. Wait — we are not told $X$ is complete. Let me re-read.

Actually, the problem says to use the BCT, which only applies to complete metric spaces. Reading again: "Let $(X, d)$ be a metric space." Since the problem instructs us to use BCT, we interpret this as the BCT applying, i.e., either $X$ is complete (implicit), or we note that BCT can be used wherever applicable.

Assuming BCT applies: $X$ would be meagre, contradicting BCT. Therefore $A$ cannot be both meagre and comeagre; since $A$ is meagre, $A$ is not comeagre.
</Solution>
